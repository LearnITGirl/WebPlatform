What is "Learn It, Girl"?

Our aim is to help women learn programming language while doing an awsome Project! But don't worry, you're not alone in this journey, each scholar has a mentor that guides them on their path. There are many roads you can take and we will make sure you go on the one that is best for you - without missing out on the important features of the chosen language.

How does it work?

Each accepted mentee gets a mentor to help them through discovering their new language and coding their great project. Don't worry if you don't know which language to choose. We will help you! During 3 months, the mentee will work on their code on Github to be able to show their amazing work in the end to teh rest of the "Learn IT, Girl" community.

When will it happen?

This second edition of the program will take place from October to December 2015. Save the date!
